# 创建工作空间

创建文件夹
mkdir vuepress-demo
进入文件夹
cd ./vuepress-demo
初始化项目
npm init

说明

# 配置package.json

添加脚本，方便运行个打包




 

"scripts": {
    "dev": "vuepress dev docs",
    "build": "vuepress build docs"
  }
